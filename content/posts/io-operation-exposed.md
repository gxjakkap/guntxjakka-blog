บทความนี้ถูกเขียนขึ้นเมื่อวันที่ 18 สิงหาคม 2020

เมื่อวันที่ 25 พฤศจิกายนที่ผ่านมา ผู้ใช้ทิวตเตอร์ [@SaraAyanaputra](https://twitter.com/SaraAyanaputra/status/1331455485401153539) โพสข้อความพร้อมรูปภาพสไลด์โครงการ IO ของกองทัพ โดยมีใจความดังนี้

![enter image description here](https://i.ibb.co/QvTtgt6/Enp-Onb-Vo-AAu-STc.png)

![enter image description here](https://i.ibb.co/s6xCCwM/Enp-Onqe-VEAIyzq-T.png)

ข้อมูลที่หลุดออกมาแสดงให้เห็นว่า ขบวนการ IO มีบัญชีผู้ใช้ทวิตเตอร์มากกว่าหลักหมื่นบัญชี และใช้แอปพลิเคชั่นเฉพาะในการสั่งงาน โดยแอปพลิเคชั่นที่ใช้ในการส่งข้อความมีชื่อว่า  [Free Messenger](https://apkcombo.com/free-messenger/com.free.messengers/)  ซึ่งพัฒนาโดย Developer นามว่า win.vong นอกจากนี้เมื่อผู้ใช้ทวิตเตอร์  [@Ovaltinez](https://twitter.com/Ovaltinez/status/1331515800524529666)  ทำการถอด Source code ของแอปพลิเคชั่นดังกล่าวพบว่า มี assets ต่างๆที่ใช้ cdn จากโดเมน mhelpme.com ซึ่งตรงกับที่ผู้ใช้ทวิตเตอร์  [@pittaya](https://twitter.com/pittaya/status/1331501696304906242)  ได้ถอด Source code แอปพลิเคชั่น  [Twitter broadcast](https://apkcombo.com/twitter-broadcast/com.wimvong.broadcast/)  ออกมาได้ผลลัพธ์ที่คล้ายคลึงกัน

ข้อมูลจากทั้ง 2 ท่านทำให้ทราบว่า บริษัทที่รับทำงานให้กับกองทัพในครั้งนี้ คือ[บริษัท เอส-แพลนเนต จำกัด](https://datawarehouse.dbd.go.th/company/profile/5/0105551043907)  โดยทราบได้จากข้อมูล whois ของโดเมน  [mhelpme.com](https://who.is/whois/mhelpme.com) ซึ่งแม้ในปัจจุบัน เว็บไซต์ s-planet.co.th จะไม่สามารถเข้าถึงได้ แต่จากข้อมูลที่ถูกเก็บไว้โดย  [archive.org](http://web.archive.org/web/20200924124936/http://s-planet.co.th/) ก็มีข้อมูลที่อยู่ อีเมลติดต่อ เป็นไปในทางเดียวกัน แล้วยังทำให้ทราบว่า บริษัทนี้มีลูกค้าส่วนใหญ่เป็นหน่วยงานของรัฐ

![enter image description here](https://i.ibb.co/HYqhZmw/Enqocdj-Vk-AIt-TV3.jpg)

กลับมาที่สไลด์โครงการ จะเห็นได้ชัดว่า โครงการนี้ มีการจัดการที่เป็นระบบ แบ่งการทำงานเป็น "แกนนำ" และ "ผู้สนับสนุน"

![enter image description here](https://i.ibb.co/y5CDvkQ/Enp-Oo1-Vk-AQDjq-K.jpg)

และมีการใช้แอปพลิเคชั่น Twitter Broadcast ในการโพสข้อความทีละหลายๆบัญชีพร้อมๆกัน ซึ่งเห็นได้ชัดว่า เป็นการสิ้นเปลืองงบประมาณโดยไม่จำเป็นในการจ้างบริษัทเอกชนพัฒนาแอปพลิเคชั่นเพื่อโครงการนี้โดยเฉพาะ ซ้ำยังสร้างความเข้าใจผิดในสังคม จึงเกิดการตั้งคำถามว่า การนำงบประมาณมาเพื่อใช้ในการปล่อยข่าวสร้างภาพลักษณ์ให้กับครอบครัวๆเดียวนั้น เหมาะสมแล้วหรือไม่ และเป็นอีกเหตุผลที่ประเทศไทยควรมีการปฏิรูปกองทัพให้โปร่งใส และทำงานรับใช้ประชาชนจริงๆ มิใช่ทำงานเพื่อตั้งตนเป็นปฏิปักษ์กับประชาชน
